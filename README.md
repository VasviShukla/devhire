# DevHire 🧑‍💻

A job board for developers — browse, filter, and apply to dev roles. Built with React.

## Features
- Search jobs by role, company, or tech stack
- Filter by role type, stack, employment type, and location
- View job details and apply directly
- Post a job form

## Tech Stack
- React 18
- CSS Modules
- React Scripts (Create React App)

## Getting Started

```bash
# Install dependencies
npm install

# Run locally
npm start

# Build for production
npm run build
```

## Project Structure
```
src/
  components/
    Navbar.jsx        # Top navigation
    BrowseJobs.jsx    # Job listing + filters
    JobCard.jsx       # Individual job card
    PostJob.jsx       # Post a job form
  data/
    jobs.js           # Job listings data
  App.jsx             # Root component
  index.js            # Entry point
```
